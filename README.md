# Full Stack Trainee Journey

This repository contains all practical assignments completed during the Full Stack Development Training Program.

Each assignment is developed in its own feature branch and merged into the `main` branch through a Pull Request after review.

## Git Workflow

For every assignment:

1. Create a new branch from `main`
2. Complete the assigned task
3. Commit the changes
4. Push the branch to GitHub
5. Open a Pull Request
6. Merge the Pull Request into `main`
